# MyEbookPub

Convert content written in markdown into a polished PDF

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'my_ebook_pub'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install my_ebook_pub
